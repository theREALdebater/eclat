### Quiet Option

Normally, a command will output (to the standard output) a small amount of information about 
what it is doing, minor warnings and hints. 

The `--quiet` (or `-q`) option causes the command to omit all such information. 

In any case, major warnings and errors are still reported (to the standard error). 


### Verbose Option

The `--verbose` (or `-v`) option causes the command to output (to the standard output) 
information about what it is doing and details of everything it is dealing with.

## Hosted Platforms

On [hosted platforms](../pxcr/targets.md#plat), each command-line tool is a separate executable 
image (file) which includes all the servers it needs. Execution of the tool includes startup of 
the servers (before the main program runs) and shutdown of the servers (after the main program 
has completed).
